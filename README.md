# TrustScreen

TrustScreen — это браузерное расширение для анализа URL и обнаружения фишинговых сайтов в режиме реального времени.

## Что умеет расширение

- Автоматически анализирует открываемые сайты в режиме реального времени.
- Позволяет вручную проверять URL через всплывающее окно расширения.
- Показывает результат анализа: вердикт, оценку риска и причины.
- Блокирует переход на сайты, определённые как фишинговые.
- Использует локальный кэш известных фишинговых доменов для резервной защиты.
- Синхронизирует список фишинговых доменов с backend-сервисом.
- Поддерживает базовые настройки интерфейса:
  - светлая и тёмная тема
  - включение и отключение анализа в реальном времени

## Важное условие использования

Если вы используете TrustScreen для анализа веб-сайтов, то функция автоматического анализа в режиме реального времени считается включённой как часть логики работы расширения.

Устанавливая и используя это расширение, вы соглашаетесь с тем, что адреса посещаемых вами веб-сайтов могут отправляться в режиме реального времени с целью их анализа.

Если вы не согласны с этим условием то вы можете отключить функцию анализа в режиме реального времени в настройках расширения (после этого для анализа надо будет вручную проверять каждый сайт через меню расширения).

## Установка

1. Скачайте или клонируйте этот репозиторий.
2. Откройте Chrome и перейдите на страницу `chrome://extensions/`.
3. Включите **Режим разработчика**.
4. Нажмите **Загрузить распакованное расширение**.
5. Выберите папку `/client` в этом проекте.

После этого иконка расширения появится в панели браузера.

## Как использовать

### Автоматический анализ

- Откройте любой сайт.
- TrustScreen автоматически отправит URL текущей страницы на анализ.
- Если сайт будет определён как фишинговый, расширение может перенаправить вас на страницу предупреждения.

### Ручная проверка

1. Нажмите на иконку расширения TrustScreen.
2. Введите URL вручную или используйте URL текущей вкладки.
3. Нажмите **Проверить URL**.
4. Ознакомьтесь с вердиктом, оценкой и причинами.

## Возможные вердикты

- `safe` — сайт выглядит безопасным
- `suspicious` — сайт вызывает подозрение и требует осторожности
- `phishing` — сайт считается опасным

## Используемые разрешения

- `tabs` — для чтения активной вкладки и обновления заблокированных вкладок
- `storage` — для сохранения настроек и кэша результатов
- `alarms` — для периодической синхронизации списка доменов
- `<all_urls>` — для мониторинга и анализа посещаемых сайтов

## Уведомление о конфиденциальности

TrustScreen создан специально для анализа веб-сайтов. Поэтому URL посещаемых страниц могут передаваться на настроенный сервер анализа в режиме реального времени.

Если вы не согласны с таким поведением, не устанавливайте и не используйте расширение или вы можете отключить функцию анализа в режиме реального времени в настройках расширения (после этого для анализа надо будет вручную проверять каждый сайт через меню расширения).
